css居中布局

发表于 2020-03-20 分类于 CSS

一、水平居中

  1. 利用 text-align: center 可以实现在块级元素内部的行内元素水平居中

  2. 如果块级元素内部包着也是一个块级元素，可以先将其由块级元素改变为行内块元素，再通过设置行内块元素居中以达到水平居中

  3. 将该块级元素左右外边距margin-left和margin-right设置为auto
  4. 使用table+margin先将子元素设置为块级表格来显示（类似），再将其设置水平居中，display:table在表现上类似block元素，但是宽度为内容宽
  1. 使用absolute+transform，先将父元素设置为相对定位，再将子元素设置为绝对定位，向右移动子元素，移动距离为父容器的一半，最后通过向左移动子元素的一半宽度以达到水平居中。(定宽就是margin-left;不定宽就transfrom)

  1. 通过flex+justify-content

  2. 通过flex+margin , 通过flex将父容器设置为为Flex布局，再设置子元素居中。

  3. 浮动元素的居中

    • 通用方法：通用办法flex布局(不管是定宽还是不定宽)利用弹性布局(flex)的justify-content属性，实现水平居中

    • 定宽：通过子元素设置relative + 负margin,

    • 不定宽：通过父子容器都相对定位，要清除浮动，给外部元素加上float。这里的父元素就是外部元素

二、水平垂直居中

  1. 使用绝对定位和负边距（已知高度和宽度）

  2. 绝对定位和margin：auto

  3. 绝对定位和transform

  4. flex布局

  5. flex/grid和margin：auto

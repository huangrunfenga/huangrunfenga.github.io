Hexo+Next 博客搭建之旅

发表于 2019-09-01 更新于 2019-09-10 分类于 Hexo

前言

  1. 先讲一下自己搭博客的目的：

    一个是在平时学习的过程中，会去网上看一些大牛的技术文章，对自己帮助很大，就想能有自己的一个博客，能有属于自己的一片栖息之所，把平时的一些学习笔记或者生活点滴记录下来，让生活有点仪式感；另一个是，这里能逼迫自己去学东西，去记录，去思考，这很重要。

  2. 给自己定一个小目标：

    坚持每天都要学习一点东西，坚持每天写博客，始终相信滴水能穿石，技术改变人生。

正文

一，为什么是Hexo

​ 因为比较容易上手，能很快的搭建起来，就如官方文档所说 “Hexo 是一个快速、简洁且高效的博客框架。Hexo 使用 Markdown（或其他渲染引擎）解析文章，在几秒内，即可利用靓丽的主题生成静态网页。”在此之前也有想过用vue+node搭建，但是太菜了，尝试用VuePress搭都搞出来很多问题，所以就退而求其次了，先用hexo其他等后面再弄！！！

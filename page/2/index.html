async await

发表于 2020-03-20 分类于 ES6

async

  1. async用来申明一个函数是异步的

  2. async函数返回的是Promise对象，如果在函数中 return 一个直接量，async 会把这个直接量通过 Promise.resolve() 封装成 Promise 对象，如果有返回值就是Promise.resolve(‘**’)，如果没有返回值就是Promise.resolve(undefined)

  3. 因为返回值是Promise对象，所以要拿到resolve出来的值得使用then()方法

  4. 所以在没有 await 的情况下执行 async 函数，它会立即执行，返回一个 Promise 对象，并且，绝不会阻塞后面的语句。这和普通返回 Promise 对象的函数并无二致

块级元素与行内元素

发表于 2020-03-20 分类于 CSS
  1. 块级元素
    • 各占一行，垂直方向排列
    • 默认宽度是父元素的100%
    • 可以包含块级元素和行内元素
    • 可以设置width height padding border margin
  2. 行内元素
    • 行内元素会在一条直线上排列（默认宽度只与内容有关），都是同一行的，水平方向排列。
    • 默认宽度只与内容有关，比如文字，图片
    • 只能容纳文本或者其他行内元素
    • width height 不可设置，但是margin padding border可以设置

浮动与BFC

发表于 2020-03-20 分类于 CSS

一、BFC定义

BFC(Block formatting context)直译为”块级格式化上下文”。它是一个独立的渲染区域，只有Block-level box参与（在下面有解释）， 它规定了内部的Block-level Box如何布局，并且与这个区域外部毫不相干。

二、触发条件

  1. 根元素
  2. float的值不为none
  3. overflow的值不为visible
  4. display的值为inline-block\table-cell\table-caption
  5. position的值为absoult或者fixed
